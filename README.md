## 控制器規格與線路

### 奧雄直流無刷超靜音控制器

![控制器線路接頭標示](images/controller.jpg)
> 圖1：各接頭位置與線色標示

### 基本規格

| 項目 | 規格 |
|------|------|
| 製造商 | 南京奧雄電子有限公司 |
| 型號標示 | AXDZ20230307 |
| 類型 | 直流無刷超靜音控制器 |
| 欠壓 | 29V / 40V |
| 功率 | 450W |
| 剎車方式 | 低電平 |
| 相位 | 120度 |

### 接頭線路標示

| 接頭名稱 | 說明 |
|----------|------|
| 報警器 | 兩個紅色接頭 |
| 儀表線 | 黑色扁平多芯接頭 |
| 剎車斷電 | 剎車時切斷馬達電源 |
| 大燈供電 | **紅正黑負**（接大燈時注意極性） |
| 剎車燈 | 剎車時點亮尾燈 |

> ⚠️ **注意**：大燈供電接頭紅線為正極、黑線為負極，接反會損壞燈具。

## 儀表設定

### 奧雄 NJAX-T 儀表

- **適用車型**：48V 電動滑板車
- **通用範圍**：36V / 48V 通用款

#### 參數設定

| 參數 | 數值 | 說明 |
|------|------|------|
| P1 | **41.5** | 已確認設定值 |

> 💡 進入設定方式：長按儀表按鍵進入 P 參數設定模式，依序調整。
